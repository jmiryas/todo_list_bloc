# Todo List BLoC

Sebuah aplikasi todo list sederhana dengan menerapkan BLoC pattern. Fitur: menambah, mengedit, menyelesaikan dan menghapus todo.

## Dependensi

- equatable 2.0.3
- flutter_bloc 7.3.1
- date_time_picker 2.1.0
- shared_preferences 2.0.8
- flutter_colorpicker 0.6.0
